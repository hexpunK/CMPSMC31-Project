Math Thing Help v0.1

By Laura Goold, Jake Ruston & Jordan Woerner

Formula

The application currently supports the following functions:

+, -, *, / - Default mathematical operators for addition, subtraction, multiplication and division.

Customizability

From - The value to start from when drawing the graph. The default is -10.

To - The value to end at when drawing the graph. The default is 10.

Increment - The value to increase up in when drawing the graph. For example, a value of 1 will mean that drawing a graph between 0 and 5 will plot points of 1, 2, 3, 4 and 5.
