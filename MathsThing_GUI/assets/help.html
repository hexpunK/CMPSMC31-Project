Math Thing Help v0.1

By Laura Goold, Jake Ruston & Jordan Woerner

Mathematical*

The application currently supports the following functions:

+, -, *, / - Default mathematical operators for addition, subtraction, multiplication and division.

floor() - Gives the largest integer that is less than or equal to the input.

ceil() - Gives the smallest integer that is greater than or equal to the input.

round() - Rounds the input to the nearest integer.

sqrt() - Finds the square root of the input.

fact() - Finds the factorial of the input.

log() - Calculates the logarithm of the input, to base 10.

ln() - Calculates the natural logarithm of the input.


Trigonometry*

sin() - Calculates the sine function of an angle.

cos() - Calculates the cosine function of an angle.

tan() - Calculates the tangent function of an angle.

sinh() - Calculates the hyperbolic sine function of an angle.

cosh() - Calculates the hyperbolic cosine function of an angle.

tanh() - Calculates the hyperbolic tangent function of an angle.


Constants

e - An approximation of Euler's number to 15 digits.

pi - An approximation of pi to 15 digits.


Customizability

From - The value to start from when drawing the graph. The default is -10.

To - The value to end at when drawing the graph. The default is 10.


*Note: All functions displayed with brackets must have their parameters passed inside the brackets to work.
